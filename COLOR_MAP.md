# Color Mapping — Unified Olive/Khaki Palette

## Allowed Colors (on-brand)
These are the ONLY colors allowed in the app:

### Primary (Olive) — main brand
- primary-50 through primary-950 (olive green)
- Used for: buttons, links, active states, accents

### Status Colors (muted military tones)
- green-500/600/700 → **KEEP** for success/active/online states
- red-500/600/700 → **KEEP** for errors/destructive/critical only
- amber-500/600/700 → **KEEP** for warnings/pending
- yellow-500/600 → REPLACE with amber (too bright)

### Neutrals
- gray-* → KEEP for disabled, muted, secondary

### REMOVE (off-brand)
- rose-* → replace with red-*
- emerald-* → replace with green-*
- violet/purple → replace with primary-*
- pink → replace with red-*
- orange-* → replace with amber-*

## Color Rules
1. green = success, active, positive ONLY
2. red = error, danger, destructive ONLY
3. amber = warning, pending, attention
4. primary (olive) = brand, interactive, selected
5. gray = disabled, muted, borders
6. NO random accent colors — everything fits the 5 above
